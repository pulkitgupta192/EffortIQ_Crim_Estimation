❓ FAQ

Q: Can I use multiple AI providers?
A: Yes! Configure all of them in Settings, switch between them anytime.

Q: Where are my API keys stored?
A: In C:\Users\YourName\.effortiq\config.json (encrypted)

Q: Can I process 1000s of items?
A: Yes! Process in batches of 50-100 for best speed.

Q: Does it work offline?
A: Partially - local operations work, but AI/Jira need internet.

Q: How do I update the app?
A: Re-run npm install, then npm start or rebuild installer.

Q: Can I run on Mac/Linux?
A: Code is compatible, but not tested. Would need build config changes.
