About EffortIQ

Your intelligent effort estimation companion

EffortIQ v1.0.0

A powerful desktop application for intelligent effort estimation and bulk Jira ticket creation.

Features:

  • ✅ Excel file upload with CRIM data
  • ✅ AI-powered effort estimation (OpenAI, Azure, Gemini, Local)
  • ✅ Bulk Jira ticket creation
  • ✅ Fetch-only mode for estimation without ticket creation
  • ✅ Graphical UI: donut charts, expandable rows, exports

Technology Stack:

  • ⚡ Electron - Cross-platform desktop app
  • 📊 XLSX - Excel file parsing
  • 🤖 Multiple AI providers integration
  • 🔗 Jira REST API integration
